Dự án: JPTaxi (ITSSJP1-20252-Thu1-JPTaxi)
Mô tả: Hệ thống kết nối tài xế nói tiếng Nhật với người Nhật cần dịch vụ taxi tại Hà Nội.
Phiên bản tài liệu: 1.0
Ngày: 10/05/2026
1. Giới thiệu
Tài liệu này hướng dẫn thiết lập môi trường phát triển cho dự án JPTaxi. Dự án bao gồm:
    - Backend: Xử lý logic, API (đăng ký, đăng nhập, đặt xe...).
    - Frontend: Giao diện người dùng.
    - Database: Lưu trữ dữ liệu người dùng, chuyến đi, tài xế.
Mục tiêu sprint: Hoàn thành cơ sở dữ liệu, môi trường dev, chức năng đăng ký/đăng nhập, xem thông tin tài khoản, và matching chuyến đi.

2. Yêu cầu hệ thống (Prerequisites)
Phần mềm yêu cầu:
    FE: HTML, CSS
    BE: SpringBoot (Java)
    DB: PostgreSQL
    Môi trường phát triển: Tuỳ cá nhân (Visual Studio Code, Eclipse IDE)
    Lưu trữ code: GitHub

3. Cài đặt công cụ cơ bản
3.1. Git
Windows: Tải từ https://git-scm.com/download/win
macOS: brew install git
Linux: sudo apt install git
Cấu hình:
Bash
git config --global user.name "Tên của bạn"
git config --global user.email "email@example.com"
3.2. Node.js (nếu dùng JavaScript/TypeScript)
Tải từ https://nodejs.org (chọn LTS).
Kiểm tra:
Bash
node --version
npm --version
3.3. Docker (Khuyến nghị mạnh)
Tải Docker Desktop: https://www.docker.com/products/docker-desktop/

4. Clone Repository
Bash
git clone https://github.com/AIVIETNAM-AIO-MaNgocThang/ITSSJP1-20252-Thu1-JPTaxi.git
cd ITSSJP1-20252-Thu1-JPTaxi
Cấu trúc thư mục hiện tại:
/backend — Code server
/frontend — Code client
/database — Script DDL, migration
/docs — Tài liệu
/package — Package/config chung (nếu có)

5. Thiết lập Backend
Bước 1: Vào thư mục backend
Bash
cd backend
Bước 2: Cài dependencies
Bash
npm install
# hoặc
yarn install
# hoặc pip install -r requirements.txt (nếu Python)
Bước 3: Tạo file môi trường .env
Copy file mẫu (nếu có .env.example) và chỉnh sửa:
env
PORT=5000
NODE_ENV=development

# Database
DB_HOST=localhost
DB_PORT=5432
DB_NAME=jptaxi_dev
DB_USER=postgres
DB_PASSWORD=yourpassword

# JWT
JWT_SECRET=your_super_secret_key_here
JWT_EXPIRES_IN=1d

# Other
GOOGLE_MAPS_API_KEY=...
Bước 4: Chạy migration / seed database (sẽ hướng dẫn chi tiết ở phần Database).
Bước 5: Khởi chạy
Bash
npm run dev
# hoặc npm start

6. Thiết lập Frontend
Bash
cd ../frontend
npm install
# Tạo .env nếu cần (API_URL=http://localhost:5000)
npm run dev
Frontend thường chạy ở http://localhost:3000 (Vite/React) hoặc cổng tương ứng.

7. Thiết lập Database
Option 1: Dùng Docker (Khuyến nghị)
Tạo file docker-compose.yml ở root project (nếu chưa có):
YAML
version: '3.8'
services:
  db:
    image: postgres:16
    restart: always
    environment:
      POSTGRES_DB: jptaxi_dev
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
Chạy:
Bash
docker-compose up -d db
Option 2: Cài trực tiếp
PostgreSQL: https://www.postgresql.org/download/
Tạo database jptaxi_dev
Chạy script trong thư mục /database.

10. Troubleshooting thường gặp
Port already in use: Kill process đang dùng port (npx kill-port 5000)
Database connection refused: Kiểm tra Docker có chạy không, hoặc credential trong .env
Module not found: Chạy npm install lại
Permission error: Chạy với quyền Administrator (Windows) hoặc sudo (macOS/Linux)

11. Lưu ý quan trọng
